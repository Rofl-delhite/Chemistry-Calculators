Step 3: Finally, the position of the chemical element in the periodic table will be displayed in the new window.

What is Meant by Periodic Table?

In chemistry, a periodic table is a table consisting of chemical elements which are arranged in the order of increasing atomic number. The elements with the same properties are arranged in the same column known as groups, whereas the elements with the same number of electrons shells are arranged in the same row known as the period. The table is called a periodic table as the elements are lined up in the periods or cycles.
